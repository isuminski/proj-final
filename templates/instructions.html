State Calculator:
Any thermodynamic state is defined by a unique set of state variables. In
the case of water, two variables are needed to define the state, and the rest can
be looked up, calculated, or interpolated from a series of tables. This tool is
designed to make that process easier. On the 'State Calculator' page, a series of
text inputs corresponding to state variables are available. On this calculator
any of the enabled inputs can be used to search for a thermodynamic state.
Details for each search combination are found below:

Valid Description Valid for:
T + P Isothermal search with specified pressure Tmin < T < Tmax
Pmin < P < Tmax
T + {h|s} Iterated isothermal search to find h or s
as specified. Will interpolate under
saturation curve. 		Tmin < T < Tmax
{h|s}Tmin < {h|s} < {h|s}Tmax
P + {h|s} Iterated isobaric search to find h or s
as specified. Will interpolate under
saturation curve. 		Pmin < P < Pmax
{h|s}Pmin < {h|s} < {h|s}Pmax
x + {P|T} Search for mixture under saturation
curve by interpolation. 		{T|P}min < {P|T} < {T|P}SatMax
0 < x < 1
Invalid Description
h + s Though in theory this search is performable through double itertion,
the computational requirements went beyond the necessary
scope of the project.
x + {h|s} Likewise, this search can be performed with some extra computations
but they went beyond what was required for this project


Cycle Calculator: (not implemented)
This function allows for four states to be combined into a simple four-stage
Rankine power cycle, and returns the efficiency and power output of the cycle.
The inputs are the same as for the State Calculator (two state variables per state)
but if they are not entered in the specified order, the calculator will return
an error.
