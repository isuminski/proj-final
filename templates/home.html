Thermodynamics Water State/Rankine Cycle Calculator!
Using the NIST Thermophysical Webbooks.

About:
This tool was created as a final project for SI 507 - "Intermediate Programming
using Python" in the Winter 2020 Term at the University of Michigan. It requires
Python3, as well as the Flask, sqlite3, and Beautiful Soup packages to run. The program
is designed to streamline the process of searching the NIST Thermophysical online
webbooks (an online tabulation of thermodynamic properties) to help solve problems
as might be found in an undergraduate engineering thermodynamics course.

program, please see the instructions page. To gain some understanding of the layout
and functionality of the database this tool is designed to search, you may visit the
